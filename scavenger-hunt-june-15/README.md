# Cousin Camp Olympics — Udvar-Hazy Scavenger Hunt (June 15)

Four age-tuned scavenger hunts for the Steven F. Udvar-Hazy Center, with a check-off website and printable sheets.

## Files
- **index.html** — the interactive website. Four campers, tap to check items off, progress saves automatically on each kid's device.
- **Forrest_Hunt.pdf / Evrin_Hunt.pdf / Geneva_Hunt.pdf / Elsie_Hunt.pdf** — print these (one per kid). 20 items each (~2 pages).
- **ANSWER_KEY.pdf** — for the grown-ups. Keep hidden!

## Put the website online (GitHub Pages)
1. Create a new GitHub repo (e.g. `cousin-camp-hunt`).
2. Upload `index.html` to it.
3. Repo **Settings → Pages → Source: Deploy from a branch → main / root → Save**.
4. After a minute, your site is live at `https://<your-username>.github.io/cousin-camp-hunt/`.
5. Share the link (or a QR code) with the kids' devices.

Note: each kid's checkmarks save in their own browser (localStorage), so everyone should use their own phone/tablet.

## The hunts
- **Forrest (8) — Junior Aviator:** big, visual spot-its.
- **Evrin (11) — Flight Engineer:** named aircraft + read-the-placard.
- **Geneva (11) — Navigator:** different aircraft, same difficulty as Evrin.
- **Elsie (13) — Mission Commander:** riddle clues + research questions.

Medal: most items found wins the day. (Equal counts — 10 each — so it's a fair race.)
